# movies_redis

A Django project that demonstrates caching of database queries using redis. 

### Dependencies:
Django==1.11.6
django-debug-toolbar==1.8
django-redis==4.8.0
pytz==2017.2
redis==2.10.6
sqlparse==0.2.4

### Set up:

1. Clone the repository
2. Activate virtualenv and install dependencies
3. Run database migrations
